# Mis Herramientas de Desbloqueo (ZonaApps)

Aquí guardo los enlaces directos y configuraciones para saltar bloqueos regionales.

## 1. Enlaces a programas de GitHub
* [GoodbyeDPI para Windows](https://github.com) -> Descargar desde la sección "Releases".

## 2. Configuración de DNS Rápidas
Si el bloqueo es por DNS, cambiar en el sistema a:
* **DNS Primario:** `1.1.1.1` (Cloudflare)
* **DNS Secundario:** `8.8.8.8` (Google)
